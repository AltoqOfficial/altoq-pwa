# Strategic Review Protocol

**Participants:** Product Manager, Business Strategist, Head of Growth.
**Trigger:** Monthly or when a major competitor update occurs (e.g., new JNE feature).

## Steps

1. **Market Scan (Head of Growth)**
   - Check JNE Voto Informado site, capictive site and votabienperu.
   - Check Competitor Apps.
   - _Action:_ List any new features that threaten our value prop.

2. **Performance Review (Business Strategist)**
   - Review User Acquisition Cost (CAC).
   - Review Retention Rate.
   - _Action:_ Flag if we are burning cash without growth.

3. **Roadmap Adjustment (Product Manager)**
   - Based on inputs, decide:
     - **Kill:** Features that no longer differentiate us.
     - **Keep:** Core features (Match).
     - **Add:** New opportunities (e.g., WhatsApp integration).

4. **Output**
   - Update `implementation_plan.md` or `roadmap.md` with new priorities.
